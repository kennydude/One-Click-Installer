Installing the 1-Click Installer
Unfortunately Ubuntu doesn't have a 1-click installer pre-installed, however a lot of people would find it useful. If you want to install, open up Terminal (Go to the Launcher and type "Terminal"). Now a scary box like thing will appear, in the box copy and paste: 
sudo add-apt-repository ppa:headbangerkenny/ppa
sudo apt-get update
sudo apt-get install one-click-installer
When that has finished, you should be able to open the file above.
